# Bài 2. Modern Big Data Stack

Ngăn xếp Big Data hiện đại là tập hợp các lớp công nghệ giúp dữ liệu đi từ nguồn vào đến giá trị đầu ra một cách trơn tru.

## 1. Các lớp chính
- Nguồn dữ liệu.
- Thu thập dữ liệu.
- Lưu trữ dữ liệu.
- Xử lý dữ liệu.
- Phân tích dữ liệu.
- Ra quyết định và hành động.

## 2. Sáu khả năng cốt lõi của stack hiện đại
1. Khám phá và điều hướng dữ liệu đa nguồn.
2. Hiệu suất cực cao, đưa phân tích đến gần dữ liệu.
3. Xử lý dữ liệu phi cấu trúc như ảnh, video, text.
4. Xử lý dữ liệu đang chuyển động, đặc biệt là streaming.
5. Thư viện phân tích và machine learning phong phú.
6. Tích hợp, quản trị, chất lượng và bảo mật.

## 3. Các thành phần điển hình
- Data sources: RDBMS, JSON, logs, text, images.
- Data ingestion: batch hoặc stream, ví dụ Kafka, NiFi, Sqoop, Flume.
- Data storage: HDFS, object storage, data lakes.
- Data processing: Spark, MapReduce, Flink, Storm.
- Data analytics: mô tả, chẩn đoán, dự đoán, đề xuất.
- Decision/action: dashboard, automation, báo cáo.

## 4. Kết luận
Stack hiện đại giúp doanh nghiệp không chỉ lưu dữ liệu, mà còn tổ chức luồng dữ liệu thành một hệ sinh thái phục vụ phân tích và hành động.
