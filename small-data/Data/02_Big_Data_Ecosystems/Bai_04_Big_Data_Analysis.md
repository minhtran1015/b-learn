# Bài 4. Phân tích Big Data

Phân tích Big Data là quá trình kiểm tra các tập dữ liệu lớn và đa dạng để tìm mẫu ẩn, mối tương quan, xu hướng thị trường và sở thích của khách hàng.

## 1. Hiểu đúng về phân tích dữ liệu lớn
- Không phải cứ nhiều dữ liệu là sẽ có nhiều trí tuệ.
- Nếu thiếu kiến trúc hệ thống, mô hình phân tích và hiểu biết miền ứng dụng, dữ liệu lớn chỉ trở thành tiếng ồn và chi phí.

## 2. Mục tiêu của phân tích
- Phát hiện quy luật từ dữ liệu thô.
- Kiểm định giả thuyết bằng quan sát và thử nghiệm.
- Chuyển dữ liệu thành thông tin, rồi thành tri thức và hành động.

## 3. Quy trình phân tích
1. Thu thập và truy cập dữ liệu.
2. Nhào nặn dữ liệu.
3. Tích hợp và biểu diễn quan hệ.
4. Làm sạch và lọc.
5. Giả thuyết, truy vấn, phân tích và mô hình hóa.
6. Thấu hiểu, lặp lại và khám phá.

## 4. Lưu ý về đạo đức
- Bảo vệ dữ liệu.
- Tuân thủ nguyên tắc thống kê.
- Không trình bày kết quả gây hiểu lầm.

## 5. Kết luận
Phân tích là cầu nối giữa hạ tầng dữ liệu và quyết định. Dữ liệu chỉ tạo giá trị khi được làm sạch, mô hình hóa và diễn giải đúng cách.
